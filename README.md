\# To-Do List PHP



Petit projet d'entraînement en PHP.

Permet d'ajouter, cocher et supprimer des tâches via les sessions.



\## Fonctionnalités

\- Ajouter une tâche

\- Marquer comme faite / à faire

\- Supprimer une tâche

\- Persistance avec `$\_SESSION`



\## Installation

1\. Copier le dossier dans `htdocs/`

2\. Lancer Apache (XAMPP)

3\. Accéder à \[http://localhost/ton-dossier]
